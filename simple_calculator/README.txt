This a toy project from ChatGPT inquiries.

simple_calculator/
│-- calculator/           # Main package
│   ├── __init__.py      # Makes it a package
│   ├── operations.py    # Defines arithmetic functions
│   ├── calculator.py    # Main calculator logic
│-- tests/                # Unit tests
│   ├── __init__.py      # Makes it a test package
│   ├── test_operations.py  # Tests for operations
│-- scripts/              # Scripts to run the calculator
│   ├── run.py           # CLI interface
│-- requirements.txt      # Dependencies (if needed)
│-- README.md             # Project description
│-- .gitignore            # Files to ignore in Git

How to run the program 
	> python -m scripts.run in the main folder simple_calculator 
	will add setup.py, so the calculator can be installed as a package
